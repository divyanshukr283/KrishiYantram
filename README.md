# 🌾 Krishi Yantram

An AI-powered Smart Agriculture Assistant built using Streamlit, Machine Learning, and Gemini AI.

## Features

- Crop Recommendation System
- Fertilizer Recommendation System
- AI Farming Assistant Chatbot
- PDF Advisory Report Generation
- Soil & Weather Analysis
- Interactive Dashboard

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Gemini AI
- Pandas
- ReportLab

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
